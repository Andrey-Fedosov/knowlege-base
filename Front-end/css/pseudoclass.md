# pseudoclass

использование псевдо классоов

наиболее часто используются следующие псевдоклассы:

* first-child;	
* last-child;
* nth-child(n);
* hover;
* visited;
* active;

## first-child
применяется для создания стилей первому дочернему элементу (к примеру - первому номеру из списка);

## last-child
